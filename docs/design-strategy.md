## 1. 베트남 소비자 심리 & 행동 패턴

### 1-1. 핵심 인사이트: "공유하는 이유"

베트남 Gen Z가 퀴즈 결과를 공유하는 3대 동기:

| 순위 | 동기 | 설명 | 설계 반영 |
|------|------|------|----------|
| 1 | **지위 과시 (Status)** | 내가 어떤 타입인지 보여주고 싶음 | 결과 카드가 "프리미엄"하게 보여야 함 |
| 2 | **정체성 표현 (Identity)** | "나는 Bright Spring이야"처럼 라벨을 소유 | 12시즌 이름이 기억에 남고 자랑스러워야 함 |
| 3 | **FOMO (놓치기 싫음)** | 친구들이 다 하고 있으면 나도 해야 함 | 소셜 피드에서 결과가 눈에 띄어야 함 |

### 1-2. 신뢰 구축 순서 (베트남 소비자)

```
1위: 친구/동료 추천 (Word of Mouth)
2위: Facebook 뷰티 그룹 리뷰 (Dep Chanh Sa 140만, Nghien Duong Da 56만)
3위: KOL/인플루언서 (69% Gen Z가 영향받아 구매)
4위: 브랜드 평판
5위: 전문가/피부과 의사
```

→ **우리 전략**: "XX명이 이 테스트를 완료했습니다" 소셜 프루프 + 결과 공유의 자연스러운 바이럴이 1위/2위 신뢰 채널을 동시에 작동시킴

### 1-3. 발견 경로 (Discovery Funnel)

```
TikTok (인지) → Facebook 그룹 (검증) → 퀴즈 참여 → Zalo/Messenger (사적 공유) → 친구 참여
     ↑                                                    ↓
     └──────────────── 사이클 반복 ──────────────────────────┘
```

**플랫폼별 역할**:
- **TikTok** (4,090만 유저): 인지 & 바이럴 확산. 결과 화면 녹화 → 숏폼 콘텐츠
- **Facebook** (8,590만): 검증 & 토론. 뷰티 그룹에서 "이거 해봤어?" 게시
- **Zalo** (7,900만 MAU): **사적 공유의 핵심**. "나 이거 나왔어!" → 1:1 전파
- **Instagram** (1,060만): 스토리에 결과 카드 공유 (시각적 브랜딩)

### 1-4. 시각 취향 (연구 기반)

베트남 유저가 선호하는 웹 디자인:
- **높은 대칭성** + **낮은 복잡성** = 깔끔하고 정돈된 레이아웃
- **중-고 채도 색상** (파스텔 X, 네온 X, 리치한 색감 O)
- **블루가 가장 선호되는 색조** (범용)
- **구조 > 색상**: 레이아웃이 특정 색상보다 더 중요

→ 현재 gold/rose 그라디언트 테마는 적절. 다만 결과 페이지에서 시즌별 컬러가 풍부하게 표현되어야 함.

### 1-5. 브랜딩 & 언어

- **영어 브랜드명 + 베트남어 콘텐츠** = 베트남 소비자가 기대하는 조합
- 영어명 = 모던/프리미엄/국제적 신호
- 베트남어 UX = 접근성과 이해도
- **한국 컨셉 크레딧**: "한국식 12시즌 퍼스널컬러 분석 기반" → K-뷰티 신뢰 활용

## 2. 고객 경험 우선 설계 (CX-First Design)

### 2-1. 화면 흐름 재설계

```
[인트로] → [트랙 선택] → [AI 또는 퀴즈] → [로딩] → [결과 헤더] → [상세 결과] → [공유/비교] → [제품 추천]
                                                         ↑
                                                    감정적 피크
                                                  (여기서 공유 유도)
```

**핵심 원칙**: 결과 공개 = 감정적 피크. 이 순간에 공유 버튼이 가장 먼저 눈에 들어와야 함.

### 2-2. 인트로 화면 개선

**현재 문제**: 가치 제안이 "12 câu hỏi"와 "AI phân tích"에만 의존.

**개선안**:

```
┌─────────────────────────────────────┐
│    Personal Color Test              │ ← 영어 브랜드 뱃지 (크레디빌리티)
│                                     │
│    Trắc Nghiệm                      │
│    Màu Sắc Cá Nhân                  │ ← 큰 타이틀
│                                     │
│    Phương pháp phân tích 12 mùa     │
│    theo chuẩn Hàn Quốc             │ ← "한국 12시즌 분석 기반"
│                                     │
│    🌸 Xuân  🌊 Hạ  🍂 Thu  ❄️ Đông  │
│                                     │
│    ╔══════════════════════════╗      │
│    ║  📷 AI Phân tích ảnh    ║      │ ← AI 트랙 (강조)
│    ║     10 giây — Chính xác  ║      │
│    ╚══════════════════════════╝      │
│    ┌──────────────────────────┐      │
│    │  📝 Trắc nghiệm 12 câu │      │ ← 퀴즈 트랙
│    │     3 phút               │      │
│    └──────────────────────────┘      │
│                                     │
│    ⭐ 50,000+ người đã tham gia     │ ← 소셜 프루프 (카운터)
│                                     │
│    Miễn phí 100% · Ảnh không lưu   │
└─────────────────────────────────────┘
```

**추가 요소**:
- "XX명 참여" 카운터 (실제 또는 점진적 증가)
- "한국 12시즌 분석 기반" 부제목 → K-뷰티 신뢰

### 2-3. 퀴즈 화면 경험 개선

**Q1 최적화** (최대 이탈 지점):
- 현재 Q1 "자연 머리색" → 적절한 워밍업. 유지.
- 힌트 텍스트를 더 친근하게: "Hãy nghĩ về màu tóc khi chưa nhuộm 😊"

**Q3-Q5 이탈 방지** ("커밋먼트 갭" 구간):
- Q3에 **시각적 요소 삽입** (예: 눈 색상 질문에 간단한 컬러 서클 이미지)
- 옵션에 작은 컬러 도트 추가 → 텍스트만의 단조로움 해소

**전체 퀴즈 UX**:
- 프로그레스: "Câu 5 / 12" 미니멀 텍스트 (퍼센트 표시 X)
- 옵션 버튼: 최소 48px 높이 (현재 16px 패딩 → 약 50px, 적절)
- 화면 전환: 현재 420ms delay → 유지 (적절한 피드백 시간)

### 2-4. 로딩 화면 경험

현재 3.2초 로딩 → **감정 구축 역할**로 강화:

```
단계 1 (0.3s): "Phân tích tông màu da..." + 스킨 아이콘
단계 2 (1.2s): "Đối chiếu màu mắt & tóc..." + 아이 아이콘
단계 3 (2.2s): "Xác định mùa màu cá nhân..." + 팔레트 아이콘
→ 결과 전환 (3.2s)
```

**추가**: 각 단계에 미세한 진행감 → 기대감 구축. 퀴즈 완료 후 결과를 기다리는 동안 이탈률 낮춤.

### 2-5. 결과 화면 — 감정적 피크 설계

**결과 공개 순서** (위에서 아래):

```
1. 시즌 뱃지 + 이모지 + 이름 (감정적 피크 — 즉시 공유 가능)
   ┌──────────────────────────────────┐
   │   🌸 Spring                       │
   │   Xuân Rực Rỡ                     │
   │   Bright Spring                   │
   │   "Rực rỡ — Tươi sáng — Trong"   │
   └──────────────────────────────────┘

2. 📸 공유 카드 + 공유 버튼 (감정적 피크 직후 배치!)
   [Facebook] [Zalo] [🔗 Link] [📥 Lưu ảnh]

3. 📊 3축 분석 차트 (Cool↔Warm, Dark↔Light, Soft↔Clear)

4. 🎨 컬러 팔레트 (추천 + 피해야 할 색)

5. 💄 메이크업 가이드

6. 👗 의상 가이드

7. 💇 헤어 컬러

8. 💍 주얼리/액세서리

9. ⭐ 같은 시즌 셀러브리티

10. 🛒 맞춤 제품 추천 (어필리에이트)

11. 👫 친구와 비교하기 (바이럴 루프)

12. 🔬 다른 테스트 (cross-link)

13. 🔄 다시 하기
```

**핵심 변경**: 공유 버튼을 **결과 이름 바로 아래**로 이동 (현재는 하단). 감정적 피크 순간에 공유를 유도.

## 3. 체류 시간 극대화 전략

### 3-1. 결과 페이지 콘텐츠 깊이

현재 결과 페이지가 이미 풍부하지만, **체류 시간을 더 늘리는 추가 섹션**:

#### A. "당신의 시즌 이야기" (스토리텔링 섹션)
```
💡 Bạn thuộc nhóm Xuân Rực Rỡ — chiếm khoảng 8% dân số.
Người Bright Spring tỏa sáng nhất khi đứng bên cạnh màu sắc
tươi sáng, sống động. Đặc trưng: tương phản cao, sắc nét...
```
- 시즌별 인구 비율 (근사치) → "나는 특별하다" 감정 유발
- 스토리텔링 → 읽는 시간 +20-30초

#### B. "이 색을 입어보세요" 가상 드레이핑 프리뷰
```
┌──────────────────────────────┐
│  [골드 톤]  [실버 톤]  [로즈]  │  ← 탭 선택
│                              │
│    ◯ 골드가 당신에게 잘        │
│    어울립니다!                 │
│    피부가 더 밝아 보이고       │
│    건강한 윤기가 납니다.      │
└──────────────────────────────┘
```
- 인터랙티브 비교 → 체류 시간 +30-60초
- 탭으로 gold/silver/rose gold 비교 (주얼리 섹션 확장)

#### C. 시즌별 Do & Don't 이미지 비교
```
✅ Nên mặc              ❌ Nên tránh
[산호색 블라우스]        [카키색 재킷]
Tông ấm sáng làm         Tông đất trầm làm
bạn tỏa sáng ✨          da bạn tái nhợt 😔
```
- 시각적 비교 → 직관적 이해 + 체류 시간

#### D. "오늘의 컬러 팁" (재방문 유도)
- 요일별 다른 스타일링 팁 → 재방문 동기
- "오늘의 추천 립 컬러" 식으로 가볍게

### 3-2. 인터랙티브 요소

| 요소 | 예상 체류 시간 증가 | 구현 난이도 |
|------|-------------------|-----------|
| 팔레트 스와치 탭 → 16진수 코드 복사 | +10초 | 낮음 (현재 hover로 보임) |
| 메이크업 컬러 도트 탭 → 제품 추천 팝업 | +15초 | 중간 |
| 셀러브리티 탭 → 해당 셀럽의 스타일링 예시 | +20초 | 중간 |
| **친구와 비교하기** 기능 | **+2-5분** (새 퀴즈 유도) | 높음 |
| **결과 카드 커스터마이징** (이름 입력) | +15초 | 낮음 |

### 3-3. 스크롤 기반 애니메이션

결과 페이지의 각 카드 섹션이 스크롤 시 **페이드인 + 슬라이드업** → 콘텐츠 발견의 재미 유지. 한꺼번에 다 보이지 않고, 스크롤할수록 새로운 정보가 나타나는 느낌.

```css
/* 이미 있는 fadeUp 키프레임 활용, Intersection Observer로 트리거 */
.card { opacity: 0; transform: translateY(24px); }
.card.visible { opacity: 1; transform: translateY(0); transition: all 0.5s; }
```

### 3-4. 체류 시간 목표

| 화면 | 현재 예상 | 목표 | 전략 |
|------|----------|------|------|
| 인트로 | 15-30초 | 20-40초 | 소셜 프루프, 트랙 선택 |
| 퀴즈 (12문항) | 2-3분 | 2.5-3.5분 | 시각적 옵션, 부드러운 전환 |
| 로딩 | 3.2초 | 3.2초 | 유지 (기대감 구축) |
| **결과 페이지** | **2-3분** | **5-8분** | 콘텐츠 깊이 + 인터랙션 |
| 비교/공유 | 0-30초 | 1-3분 | 비교 기능, 카드 커스텀 |
| **전체 세션** | **~5분** | **~12-15분** | |

## 4. 바이럴 루프 설계

### 4-1. 결과 카드 이미지 (바이럴 핵심)

**이미지가 있는 퀴즈 = 공유 확률 40배.** 이것이 가장 높은 ROI 기능.

#### 결과 카드 사양

```
┌────────────────────────────────────┐  1200 × 630px
│                                    │  (OG 이미지 표준)
│  Personal Color Test               │
│                                    │
│        🌸                          │
│   Xuân Rực Rỡ                      │
│   Bright Spring                    │
│                                    │
│   "Rực rỡ — Tươi sáng — Trong"    │
│                                    │
│   ┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐       │  ← 팔레트 6색 미니
│   │  ││  ││  ││  ││  ││  │       │
│   └──┘└──┘└──┘└──┘└──┘└──┘       │
│                                    │
│   personal-color-test.pages.dev    │  ← URL (재방문 유도)
└────────────────────────────────────┘
```

**Canvas API로 생성** → 다운로드 가능 + 동적 OG 이미지로 활용

#### 12시즌별 고유 그라디언트 배경
- 각 시즌의 `gradient` 값을 카드 배경에 적용
- 소셜 피드에서 즉시 구별 가능한 시각적 아이덴티티

### 4-2. 공유 채널 설계

**현재**: Facebook + 링크 복사 (2개)
**개선**: 4개 + 카드 다운로드

```
┌─────────────────────────────────────────────┐
│ 📤 Chia sẻ kết quả                          │
│                                             │
│ ┌──────────┐ ┌──────────┐ ┌──────────┐     │
│ │ Facebook │ │   Zalo   │ │ 🔗 Link  │     │
│ └──────────┘ └──────────┘ └──────────┘     │
│                                             │
│ ┌────────────────────────────────────┐      │
│ │  📥 Lưu ảnh kết quả               │      │ ← 결과 카드 다운로드
│ │     (Chia sẻ lên Story/TikTok)    │      │
│ └────────────────────────────────────┘      │
└─────────────────────────────────────────────┘
```

**Zalo 공유 추가 이유**:
- 7,900만 MAU, 베트남 #1 메시징 앱
- 18-30세가 가장 활발한 세그먼트
- AI 카드/아바타가 Zalo에서 대규모 바이럴 사례 (370만 AI 카드, 800만 AI 아바타)
- **사적 공유 = 가장 강한 신뢰 채널** (1위 WOM)

### 4-3. 친구와 비교하기 (바이럴 계수 2배)

**Ktestone 바이럴 모델 적용**:

```
┌─────────────────────────────────────────────┐
│ 👫 So sánh với bạn bè                       │
│                                             │
│ "Bạn là Xuân Rực Rỡ — hãy xem bạn bè     │
│  của bạn thuộc mùa nào!"                   │
│                                             │
│ ┌────────────────────────────────────┐      │
│ │  🔗 Gửi link cho bạn bè           │      │
│ │     Sau khi họ làm xong,          │      │
│ │     bạn sẽ thấy kết quả so sánh!  │      │
│ └────────────────────────────────────┘      │
│                                             │
│ Hoặc nhập mùa của bạn bè:                  │
│ [Spring ▼] [Bright ▼]  [So sánh]           │
└─────────────────────────────────────────────┘
```

**비교 결과 화면**:

1. **빌드 도구/번들러 도입하지 않음** — Vanilla JS + ES Modules으로 충분
2. **프리미엄 결제 게이트 구현하지 않음 (Phase 1-2)** — 무료 유지하여 유저 확보 우선
3. **이메일 하드 게이트 하지 않음** — 결과 차단하면 공유 불가 → 바이럴 사멸
4. **팝업 광고 사용하지 않음** — 이탈률 급증, 신뢰도 하락
5. **배포하지 않음** — 사용자 지시 전까지 로컬 작업만
