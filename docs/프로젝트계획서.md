## 1. 프로젝트 개요

### 프로젝트명
**Personal Color Test (퍼스널 컬러 테스트)**
베트남 타겟 퍼스널 컬러 분석 웹 앱

### 배경 및 시장 기회

| 항목 | 내용 |
|------|------|
| 타겟 시장 | 베트남 (인구 1억, 20~35세 여성 중심) |
| 월 검색량 | **18,000+** ("test mau sac ca nhan" 관련 키워드) |
| 뷰티 어필리에이트 수수료 | Shopee 기준 **15~30%** |
| 기존 경쟁 현황 | 한국어/영어 서비스 위주, 베트남어 특화 서비스 부재 |
| 베트남 뷰티 시장 성장률 | 연 15%+ (동남아 최고 성장률) |

### 목표

- **skintypetest.com 성공 모델 재활용**: 동일 기술 스택(HTML/CSS/Vanilla JS)과 배포 파이프라인(GitHub Actions + Cloudflare Pages)을 재사용하여 개발 기간 단축
- **교차 유도 시너지**: 피부 타입 테스트 ↔ 퍼스널 컬러 테스트 간 상호 링크로 두 사이트 모두 트래픽 증가
- **수익화 구조 복제**: AdSense + Shopee 어필리에이트 모델을 그대로 적용

### 핵심 차별화 요소

| 차별화 | 설명 |
|--------|------|
| **AI 사진 분석 + 질문형 2트랙** | 사용자가 선택 가능 (AI: 10초 / 퀴즈: 3분) |
| **베트남 특화** | 베트남어 UI, 베트남 셀럽, 현지 Shopee 제품 추천 |
| **자체 HSL 분석 알고리즘** | 외부 ML 모델 의존 없음, 경량화 구현 |
| **무료 + 가입 불필요** | 즉시 테스트, 결과 공유 가능 |
| **모바일 퍼스트** | getUserMedia API로 모바일 카메라 직접 지원 |

### 4.2 점수 체계 (3축 양극성)

퀴즈와 AI 분석 모두 동일한 3축 점수 체계를 사용하여 시즌을 결정한다.

```
축 1: temp (온도)       Cool (-) ◄━━━━━━━━━━━━► (+) Warm
축 2: depth (깊이)      Dark (-) ◄━━━━━━━━━━━━► (+) Light
축 3: clarity (선명도)   Soft (-) ◄━━━━━━━━━━━━► (+) Clear
```

**퀴즈 점수 누적 방식:**
- 12문항 각 선택지마다 `temp`, `depth`, `clarity` 가중치가 부여됨
- 예: "Đen tuyen hoac nau rat dam" → `temp: -1, depth: -2, clarity: 1`
- 전체 선택지의 점수를 누적하여 최종 3축 합산 점수 산출

**시즌 결정 공식 (Affinity Formula):**

```
spring  = +temp + depth + clarity   (Warm + Light + Clear)
summer  = -temp + depth - clarity   (Cool + Light + Soft)
autumn  = +temp - depth - clarity   (Warm + Dark  + Soft)
winter  = -temp - depth + clarity   (Cool + Dark  + Clear)
```

가장 높은 affinity 점수를 가진 시즌이 결과로 선택된다.

**4시즌 특성 요약:**

| 시즌 | 온도 | 깊이 | 선명도 | 대표 키워드 |
|------|------|------|--------|-------------|
| Spring (Xuan) | Warm | Light | Clear | 밝고 따뜻, 생기 |
| Summer (Ha) | Cool | Light | Soft | 부드럽고 시원, 우아 |
| Autumn (Thu) | Warm | Dark | Soft | 깊고 따뜻, 고급 |
| Winter (Dong) | Cool | Dark | Clear | 선명하고 차가움, 강렬 |

         v

              │
              v

│     피부 45% + 머리카락 35% + 눈 20%             │
│   - 6단계 밝기 구간 → -5 ~ +5                   │

### 8.1 AI 정확도 리스크

| 리스크 | 영향 | 확률 | 대응 |
